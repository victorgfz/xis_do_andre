# Xis do André - App de Delivery

Este é um aplicativo de delivery de comida desenvolvido em Kotlin com Jetpack Compose, seguindo a arquitetura moderna recomendada pelo Google (Guia oficial de Arquitetura Android).

## Pré-requisito

* Para executar o app, é necessário inserir o `google-services.json` na pasta `/app` (Integração com Firebase)
* É necessário que existam produtos cadastrados na coleção `produtos` do Firestore
